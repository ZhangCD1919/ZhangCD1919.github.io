# 两数之和

# 题目描述

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9 所以返回 [0, 1]

# 解题思路

首先想到的方案是双层for循环，从头开始逐个相加，不过时间复杂度较高，再想。
后来想到，一层for循环，从头开始，寻找是否有target与之相减的值并定位位置。

  • 在该方案中，开始想到用indexOf查找第二个值，发现时间复杂度仍然很高，换做对象存储。

我的代码及步骤如下：

//  indexOf查找
var twoSum = function(nums, target) {
    for(let i= 0; i < nums.length; i++) {
        let n1 = nums[i];
        let n2Index = nums.indexOf(target - n1);
        if(n2Index !== -1 && n2Index !== i) {
            return [i, n2Index];
        }
    }
    return [];
};
// 对象存储查找
var twoSum = function(nums, target) {
  var numMap = {};
    for(let i= 0; i < nums.length; i++) {
        let n1 = nums[i];
        let n2Index = numMap[target - n1];
        numMap[n1] = i;
        if(n2Index !== undefined) {
            return [i, n2Index];
        }
    }
    return [];
};
