# 罗马数字转整数

# 题目描述

罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。

字符 数值
I 1
V 5
X 10
L 50
C 100
D 500
M 1000

例如， 罗马数字 2 写做 II ，即为两个并列的 1。12 写做 XII ，即为 X + II 。 27 写做  XXVII, 即为 XX + V + II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 IIII，而是 IV。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 IX。这个特殊的规则只适用于以下六种情况：

I 可以放在 V (5) 和 X (10) 的左边，来表示 4 和 9。
X 可以放在 L (50) 和 C (100) 的左边，来表示 40 和 90。  C 可以放在 D (500) 和 M (1000) 的左边，来表示 400 和 900。
给定一个罗马数字，将其转换成整数。

示例 1: 输入: "III" 输出: 3
示例 2: 输入: "IV" 输出: 4
示例 3: 输入: "IX" 输出: 9
示例 4: 输入: "LVIII" 输出: 58 解释: L = 50, V= 5, III = 3.
示例 5:输入: "MCMXCIV" 输出: 1994 解释: M = 1000, CM = 900, XC = 90, IV = 4.

# 解题思路

首先找规律：罗马数字转为对应映射数字，然后这些数字相加得到最终整数。
对应规则有什么：I V X L C D M IV IX XL XC CD CM
当我们得到一个罗马数字时，如 MCMXCIV，首先映射出数字：M CM XC IV ———— 1000 900 90 4
然后相加 1000 + 900 + 90 + 4 得到结果 1994
所以我的思路是，构造映射结构，将输入按照映射拆分，然后相加映射对应值。

我的代码及步骤如下：

var romanToInt = function (s) {
  // 罗马与数字映射
  const nMap = {
    CM: '900|',
    CD: '400|',
    XC: '90|',
    XL: '40|',
    IX: '9|',
    IV: '4|',
    M: '1000|',
    D: '500|',
    C: '100|',
    L: '50|',
    X: '10|',
    V: '5|',
    I: '1|',
  };
  let n = s;
  // 得到 '1000|900|90|4|'
  Object.keys(nMap).forEach((key) => {
    let reg = new RegExp(key, 'g');
    n = n.replace(reg, nMap[key]);
  })
  let result = n.split('|');
  // 得到[1000, 900, 90, 4],相加各项
  result = result.reduce(function (a, b) {
    return Number(a) + Number(b);
  });
  // 得到1994
  return result;
}

# 最长公共前缀

# 题目描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1:

输入: ["flower","flow","flight"] 输出: "fl"

示例 2: 输入: ["dog","racecar","car"] 输出: ""
解释: 输入不存在公共前缀。

# 解题思路

公共前缀，也就是所有项都存在的前缀，所以我首先想到把第一项拿出来，然后将第一项从前往后查找，是否余后的项里都有到某下标的字符串。
我的代码及步骤如下：

var longestCommonPrefix = function(strs) {
    //  特殊情况判断
    let str1 = strs[0];
    if(!str1) return '';
    if(strs[1] == undefined) return str1;
    // 将数组转为字符串 用正则match匹配数组字符串中【 ，+ 首项开始至某下标】的数量length，length等于数组长度-1，则匹配成功
    let strsJoin = strs.toString(), sub = 0, subStr1 = str1[sub], reg = new RegExp(`,${subStr1}`, 'g'), ret = '';
    while(sub < str1.length && strsJoin.match(reg) && strsJoin.match(reg).length === strs.length - 1) {
        ret = subStr1;
        sub ++;
        subStr1 += str1[sub];
        reg = new RegExp(`,${subStr1}`, 'g');
    }
    return ret;
};
